# Glossary

| Term | Definition |
|------|------------|
| **Coolify** | Open-source self-hosted PaaS (apps, DBs, compose stacks) |
| **Resource** | Deployed app, database, or service in Coolify |
| **Nixpacks** | Auto buildpack for many languages |
| **Magic env vars** | Coolify-injected compose variables for service wiring |
| **Sentinel** | Coolify lightweight metrics collector |
| **Traefik** | Default reverse proxy on Coolify servers |
| **Hetzner snapshot** | Point-in-time VPS disk image for rollback |
| **DOPPLER_WRAP** | Entrypoint pattern wrapping `doppler run` |
| **APP_COMMAND** | Env var to run alternate process from same image |
